## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Lucasbritx/delta-test.git
```

Entre no diretório do projeto

```bash
  cd delta-test
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```

## Stack utilizada

**Front-end:** React, Vite, TailwindCSS
